# 语音口令清单

## 标准命令

### 单窗控制

| 目标 | 打开命令 | 关闭命令 |
| --- | --- | --- |
| 左前窗（驾驶位） | 打开左前窗 | 关闭左前窗 |
| 右前窗（副驾驶） | 打开右前窗 | 关闭右前窗 |
| 左后窗 | 打开左后窗 | 关闭左后窗 |
| 右后窗 | 打开右后窗 | 关闭右后窗 |

### 全窗控制

| 打开命令 | 关闭命令 |
| --- | --- |
| 打开全部车窗 | 关闭全部车窗 |

## 同义表达

以下表达与标准命令等价，可被正确识别。

### 动作动词

| 标准动作 | 同义表达 |
| --- | --- |
| 打开 | 开启、降下 |
| 关闭 | 关上、升起 |

### 目标车窗

| 标准名称 | 同义表达 |
| --- | --- |
| 左前窗 | 左前车窗、驾驶位车窗 |
| 右前窗 | 右前车窗、副驾驶车窗 |
| 左后窗 | 左后车窗 |
| 右后窗 | 右后车窗 |
| 全部车窗 | 所有车窗、四个车窗 |

### 礼貌/连接词（自动忽略）

"请"、"帮我"、"把" 以及空格和中文标点（，。！等）会在解析前自动移除，不影响命令识别。

示例：
- "请把左前窗打开" → 等同于 "打开左前窗"
- "帮我关闭全部车窗" → 等同于 "关闭全部车窗"

## 不支持表达

以下类型的表达不会被识别为有效命令，系统会给出具体提示：

| 类型 | 示例 | 提示 |
| --- | --- | --- |
| 缺少目标 | "打开车窗" | "请指定车窗" |
| 缺少动作 | "左前窗" | "请说明要打开还是关闭" |
| 动作冲突 | "打开再关闭左前窗" | "命令有歧义" |
| 多目标 | "打开左前窗和右前窗" | "命令有歧义" |
| 部分开度 | "开一半"、"开一点"、"半开" | "不支持的命令" |
| 排除表达 | "除了左前窗都关闭" | "不支持的命令" |
| 无关内容 | "今天天气怎么样" | "不支持的命令" |
| 非车窗目标 | "打开天窗" | "不支持的命令" |

## 语音交互状态

| 状态 | 界面表现 | 用户操作 |
| --- | --- | --- |
| 未启动 (idle) | 麦克风按钮 + "点击麦克风开始语音控制" | 可点击开始 |
| 等待授权 (permission) | "等待麦克风授权…" | 浏览器弹窗处理 |
| 聆听中 (listening) | 按钮脉冲动画 + "正在聆听…" | 说出命令 / 点击停止 |
| 识别成功 (success) | 显示识别原文 + 执行结果 | 自动执行，状态回 idle |
| 识别失败 (error) | 具体错误原因 | 可重试或使用文本 |
| 不支持 (unsupported) | 按钮禁用 + 降级提示 | 使用文本或点击控制 |

## 错误类型

| 错误码 | 含义 | 用户提示 |
| --- | --- | --- |
| denied | 麦克风权限被拒绝 | "请在浏览器设置中允许后重试" |
| no-speech | 未检测到语音 | "未检测到语音，请重试" |
| aborted | 识别被取消 | "识别已取消" |
| network | 网络错误 | "网络连接失败，请重试" |
| timeout | 10 秒无结果 | "识别超时，请重试" |
| not-supported | 浏览器不支持 | "您的浏览器不支持语音识别" |

## 文本降级

文本输入框支持与语音完全相同的命令解析规则。在文本框中输入标准命令或同义表达，点击"发送"或按 Enter 即可执行。语音不可用时文本控制始终可用。
