## 3. 实现了哪些关键逻辑

- **加载进度**（`useProgress`）：Drei 的 `useProgress` 提供 `{ progress: number, active: boolean }`——`progress` 为 0–100 的加载百分比，`active` 表示是否仍在加载。组件显示格式化的百分比 + CSS 进度条（宽度过渡动画）。`active=false` 时显示"正在准备场景…"作为过渡提示。进度条使用 `transition: width 0.3s ease` 实现平滑递增，不倒退。
- **重置视角**（`ResetCamera`）：
  - 首次挂载时 `useEffect` 将 `DEFAULT_POSITION` 更新为当前相机位置（捕获实际传入的相机配置）。
  - 点击按钮时记录起始位置（`startPos`）、目标位置（`DEFAULT_POSITION`），重置 `elapsed=0`，设置 `animating=true`。
  - `useFrame` 中累加 `delta * LERP_SPEED`，计算 ease-out 进度 `1-(1-t)²`，用 `camera.position.lerpVectors(start, target, eased)` 插值位置，同时 `camera.lookAt(DEFAULT_TARGET)` 确保朝向正确。
  - 动画完成时 snap 到精确目标值并设 `animating=false`。
  - 按钮通过 Drei `Html` 的 `calculatePosition` 固定在视口右上角（`[size.width - 130, 50]`），不随 3D 场景旋转而移动。
- **视觉增强**：
  - Key light：强度提升 + 暖白色温（#fff8f0）增强车漆 PBR 反射的棚拍质感。
  - Shadow map：1024→2048 分辨率，减少阴影锯齿。
  - Ambient：0.4→0.5 提升暗部可见度。
  - ContactShadows：增大范围/模糊/距离，使地面阴影更自然。
  - OrbitControls 显式设置 `target={[0, 0.5, 0]}` 确保初始朝向正确（车辆中心）。
